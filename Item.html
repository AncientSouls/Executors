    preparation(callbackopt)

    Attention! Must be overrided.

    Starts the pre-mount preparation actions. It includes:

    • Subscribe and getting data based on query.
    • Generate names and queries for childs.
    • Call Item#prepareChild for each needed child, to generate list of not prepared childs. This list will be used to prepare childs.
    • Must call Item#prepared after all preparations are done. It will complete stage prepare and set true into Item#isPrepared. Run childs preparation and only after all childs prepared, fires callback and prepared event. This can lead to recursion of child preparation, everything is fine, the funicular itself will deal with this.
    Source:
    • item.js, line 141
    Parameters:
    Name Type Attributes Description
    callback Item~prepareCallback <optional>
    Example
    new Manager(
  class extends Item {
    preparation(callback) {
      this.data = collection.get(this.query);
      // May be your data is { _id: 'a', childs: { b: { _id: 'b' }, c: { _id: 'c' } } }
      for (var c in this.data.childs) {
        this.prepareChild(c, this.data.childs[c]);
      }
      this.prepared(undefined, callback);
    }
  }
)
